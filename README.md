# Word LaTeX to Equation Macro (Word LaTeX 公式自动转换宏)

这是一个轻量级但功能强大的 Microsoft Word VBA 宏工具。它可以自动扫描 Word 文档中的 LaTeX 公式代码，并将其一键转换为 Word 原生的专业公式格式（OMML），极大地提升了理工科论文、实验报告和工程文档的排版效率。

## ✨ 核心功能 (Features)

* **双模式完美支持**：
  * **行内公式 (Inline)**：自动识别单 `$` 包裹的公式，转换为与文字紧凑同行的内嵌公式。
  * **行间公式 (Display)**：优先识别双 `$$` 包裹的公式，转换为居中放大、独占一行的独立公式。
* **智能防错位机制 (Parity Shift Protection)**：独创的跳过逻辑，当文档中漏写 `$` 或出现意外换行时，宏会自动保护附近的普通文本，彻底解决传统替换方法容易“吞字”和“奇偶错位”的致命 Bug。
* **多行公式支持**：对于 `$$` 包裹的复杂长公式（如多行方程组、矩阵），自动将其内部的硬回车转换为软换行，确保公式完整不碎裂。
* **极速转换 (High Performance)**：采用后台静默处理机制（禁用屏幕更新），即使面对包含上百个公式的万字长文，也能在瞬间秒转完成，告别屏幕闪烁。

## 🚀 如何安装和使用 (Installation & Usage)

1. 打开 Microsoft Word，按下 `Alt + F11` 打开 VBA 编辑器。
2. 在左侧的“项目”窗口中，右键点击 `Normal`（推荐，这样所有文档均可使用），选择 **插入 (Insert) -> 模块 (Module)**。
3. 将本项目中的 `ConvertLatexToEquation.bas` 代码完整复制并粘贴到右侧的空白代码窗口中。
4. 关闭 VBA 编辑器。
5. 在 Word 文档中正常撰写带有 LaTeX 语法的公式。
6. 按下 `Alt + F8`，选择 `ConvertLatexToEquation`，点击 **运行 (Run)** 即可一键转换全文。

## 💡 进阶技巧 (Tips)
为了最大化排版效率，强烈建议在 Word 中为该宏分配一个**专属快捷键**（如 `Ctrl + Shift + M`），或者将其添加到 Word 顶部的**快速访问工具栏**中。

## ⚠️ 注意事项 (Notes)
* 本宏底层调用的是 Word 内置的公式解析引擎 (UnicodeMath)。它对主流的 LaTeX 语法（如分数、根号、微积分符号、希腊字母、上下标等）支持极佳。
* 对于极其复杂的宏包级专属 LaTeX 环境，Word 的原生引擎可能无法完美解析。
* 行内单 `$` 公式不支持跨段落（这是为了防止漏写 `$` 导致的灾难性吞字）。如果你的公式很长且包含回车，请使用 `$$` 包裹。
